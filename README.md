# Personal Portfolio

This repository contains the source code for my personal portfolio website.

The goal of this project is to present my work, projects and skills as a web developer while practicing modern web development using HTML, CSS and JavaScript.

About the Project

This portfolio was designed to be clean, responsive and visually engaging. It showcases some of the projects I have built while studying Systems Analysis and Development.

The website includes:

- A hero section introducing who I am
- An about section describing my interests and goals
- A projects section highlighting some of my work
- A skills section with the technologies I use and study
- A contact section with my social links

The interface focuses on simplicity, usability and responsive design.

Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- Font Awesome Icons
- Google Fonts

Features

- Responsive layout for mobile and desktop
- Scroll reveal animations
- Mobile navigation menu
- Language toggle (English / Portuguese)
- Minimalist and modern interface

Projects Included

Some of the projects presented in this portfolio include:

- To-Do List – A simple task management application.
- Blender Combinations Website – An interactive website created for a school project about mathematical combinations.
- Interactive Birthday Card – A creative interactive card built with web technologies.

Purpose

This portfolio was created to showcase my learning journey in web development and to demonstrate my ability to design and build modern interfaces.
